### 1、组件之间的传值、方法调用
#### 1、子组件调用方法
在父组件的json文件中注册子组件的路径
```json
{
  "usingComponents": {
    "progress": "../componets/progress/progress"
  }
}
```
直接在xml中使用就可以
```xml
<progress></progress>
```
#### 2、子组件调用父组件页面刷新方法
需要子组件获取父组件的方法
``` js
methods: {
    switch: function(e) {
        const page = getCurrentPages().pop(); // 当前所在的父组件页面
        page.onShow() // 父组件页面刷新       
    }
}
```
